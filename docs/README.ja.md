# yasuna

Godot 4向けのグラフベースのステートフルなシナリオアドオン。

🌐 [English](/README.md) | **日本語**

**yasuna**はキューを組み合わせたシナリオフローを構築するためのアドオンです。  
実行時に非同期アクションの状態保持が可能で、ゲームプレイ中の任意のタイミングでセーブ、ロードをできるよう設計されています。  
特定ゲーム向けの機能を内蔵せず、ゲームの開発者がゲーム固有のロジックを柔軟に実装・拡張する前提としています。

> [!WARNING]
> **yasuna**は開発初期段階です。  
> 現時点では本番用途はもちろん、実プロジェクトの実験的な利用も推奨されていません。  
> API、設計、UIなど、あらゆる要素に今後大きく変更が入る可能性があります。

## 機能

- 非同期なキューのサポート
- ゲーム固有のロジックを実装するカスタムキューを容易に追加可能
- `VisualShader` のようなグラフエディタでのシナリオ・フロー管理
- 実行中の任意のタイミングで状態をセーブ・ロードできるステートフルな設計
- フローの流れをリアルタイムで見られるビジュアルデバッガー
- ピュア GDScript 製
